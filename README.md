# Drivin ROR

Esta es una aplicación Ruby on Rails para gestionar rutas y viajes.

## Requisitos

- Ruby 3.4.2
- Rails 8.0.2

## Configuración

1. Clona el repositorio:

   ```sh
   git clone https://github.com/tu_usuario/drivin_ror.git
   cd drivin_ror
   ```

2. Instalar las dependencias

```sh
bundle install
```

3. Configurar la base de datos

```sh
rails db:create
rails db:migrate
```

4. Ejecutar

```sh
rails server
```
